◆ decrypt_private_key()

pades-app-main.decrypt_private_key ( )

Decrypts the encrypted private key using the provided PIN.

The PIN is being hashed with SHA-256 algorithm and then is used do decode the AES algorithm in CBC mode with the provided private key.

Returns
Decrypted private key in PEM format, or None when key has been removed from the defined path.
